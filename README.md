# Banco de Dados - Projetos de Estudo

Este repositório contém diversos projetos de banco de dados desenvolvidos para fins acadêmicos e prática de modelagem relacional, criação de tabelas, relacionamentos, consultas SQL e manipulação de dados.

## cinema

Sistema de gerenciamento de cinema contendo informações sobre filmes, sessões, salas, clientes, ingressos e vendas.

## client-system

Sistema de cadastro e gerenciamento de clientes, permitindo armazenar informações pessoais, contatos e histórico de operações.

## ecommerce

Banco de dados para uma loja virtual, contendo produtos, categorias, clientes, pedidos, pagamentos e controle de estoque.

## escola

Sistema acadêmico responsável pelo gerenciamento de alunos, professores, turmas, disciplinas e notas.

## estoque_comercial

Sistema de controle de estoque comercial, permitindo o gerenciamento de produtos, fornecedores, entradas, saídas e movimentações de estoque.

## hospital

Banco de dados hospitalar contendo informações sobre pacientes, médicos, consultas, exames, internações e convênios.

## hospital2

Versão alternativa ou aprimorada do sistema hospitalar, utilizada para testes de modelagem, consultas e relacionamentos mais avançados.

## itens_estoque

Projeto focado no controle detalhado dos itens armazenados em estoque, incluindo quantidades, localização, validade e movimentações.

## lojadb

Sistema de gerenciamento de loja com controle de clientes, produtos, vendas, pagamentos e estoque.

## lojinhadb

Versão simplificada de um sistema comercial para estudo de modelagem relacional, consultas SQL e gerenciamento de vendas.

---

## Objetivos dos Projetos

* Praticar modelagem de banco de dados relacional;
* Aplicar conceitos de chaves primárias e estrangeiras;
* Desenvolver consultas SQL utilizando JOIN, GROUP BY, HAVING e funções de agregação;
* Simular cenários reais de sistemas corporativos;
* Aprimorar conhecimentos em MySQL.

## Tecnologias Utilizadas

* MySQL
* SQL
* Docker (quando aplicável)
* MySQL Workbench
